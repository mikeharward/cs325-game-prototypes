Escape the Volcano!

  • Originality: What is original about your game? The two-player puzzle platforming that the game allows for is fairly unique, and can be used to create interesting puzzles. The rising lava also adds an element of tension and forces the players to act quickly.
  • Technical Merit: What is interesting about your game technology? The most difficult part was detecting when the players left the buttons, as phaser only allows for overlap checking. I had to test for overlap, and auotmatically reset the button when the player on it jumped, which ended up working well. The other main issue was trying to prevent collision with the buttons, as they are very prone to getting pushed around. I ended up using the level design to mitigate this, but it would have to be addressed further if the game were to become more complex.
  • Prototype Postmortem: What did you learn from this prototype? What was the easiest or hardest part of making it? Controlling two players at once was really interesting, I hadn't had to make a more complex control scheme yet, and learned a lot from that. I also had to do more collision and overlapping than the simple tests I've used so far, so learning more about how those functions work in phaser was quite interesting. I think the easy part was the rising lava, as it was fairly simple to make and implement.
